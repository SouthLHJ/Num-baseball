## 간단 소개

    대중적으로 알려진 숫자 야구 게임을 앱으로 만들었습니다.

    간단한 규칙을 설정하고 게임 진행 도중 걸린 시간과 도전 횟수를 종료 후 보이게 만들었습니다.  

## 사용 기술 및 라이브러리

- ReactNative, EXPO

## 담당한 기능 (Android & iOS)

- 숫자를 입력하는 부분과 추측한 번호의 기록을 볼 수 있는 부분이 나옴
- 같은 숫자를 입력할 수 없게 알람을 띄우게 만듦

## 깨달은 점

- ReactNative를 이용하여 ios와 android 호환 앱을 쉽게 만들 수 있다.
